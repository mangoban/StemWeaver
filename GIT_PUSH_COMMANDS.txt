# STEP-BY-STEP: Push StemWeaver to GitHub
# ========================================

# 1. Initialize Git
git init
git branch -M main

# 2. Add GitHub remote (replace with your actual username if different)
git remote add origin https://github.com/mangoban/StemWeaver.git

# 3. Add all files (respecting .gitignore)
git add .

# 4. Check what will be committed
git status

# 5. Commit with message
git commit -m "Initial commit: StemWeaver v1.1 - Professional Audio Stem Separation

- AI-powered stem separation using Meta Demucs
- Modern GUI with theme support
- CPU/GPU processing
- Multiple extraction profiles (6-stem, 4-stem, vocals-only)
- MP3/FLAC/OGG/WAV support
- CC BY 4.0 license
- AppImage distribution

Created by bendeb creations"

# 6. Push to GitHub
git push -u origin main

# 7. Optional: Create a release for the AppImage
# Go to https://github.com/mangoban/StemWeaver/releases
# Upload StemWeaver-v1.1-x86_64.AppImage as a release asset

# Done! Your code is now on GitHub
